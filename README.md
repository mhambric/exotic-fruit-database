## Exotic Fruit Database System

This repository contains the relational database design for the Exotic Fruit Storefront system.

### Technologies
- MySQL
- SQL
- Stored Procedures

### Database Design
- ER diagram describing entities and relationships
- Normalized schema (3NF)

### Key Features
- Product, Category, Customer, Order, and OrderItem tables
- Foreign key constraints
- Stored procedures for data reset and reporting

### Sample Queries
Screenshots and examples of joins, reports, and aggregations.

### How this connects to the frontend
This database was designed to support the frontend storefront and client dashboard workflows.
